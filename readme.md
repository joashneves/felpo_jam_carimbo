## Descrição e instruções
- Este é um jogo no qual o jogador precisa carimbar o maximo de papeis possivel, ele tambem precisa tomar cuidado
com os itens que não são usados juntos a um carimbo
## Controles
- Para se mover
	- W ou seta para cima (controle, tanto no analogico, quanto o +)
	- D ou seta para baixo 
	- Espaço ou click do mouse Para carimbar os papeis
